# Smart AI Glasses for Visually Impaired

## Description
An AI-powered assistive system designed to help visually impaired users detect obstacles and recognize objects in real-time.

## Features
- Real-time obstacle detection using Ultrasonic Sensors
- Object recognition using YOLOv8
- Arabic voice feedback using Text-to-Speech
- Enhanced user safety and independence

## Technologies
- ESP32-CAM
- Ultrasonic Sensors
- Python
- YOLOv8
- gTTS (Google Text-to-Speech)
